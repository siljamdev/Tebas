# Tebas Exit Codes

|Code|Meaning|
|---|---|
|0|No errors happened|
|1|CLI command argument number problem|
|2|CLI unknown command|
|8|CLI invalid argument|
|11|Problem occured while initializing app|
|12|Problem occured while initializing app|
|21|No local project can be found|
|22|Referenced template/plugin is not installed|
|23|Script not found|
|24|Config/permission problem|
|25|Problem occured while installing/uninstalling template/plugin|
|26|Problem occured on init a new project|
|30|Another exception occured|
|50|Problem occured while updating|
